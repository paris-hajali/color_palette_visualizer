# Color Palette Visualizer

Visualize color palettes with hex code inputs. Paste any text (JSON, CSS, design tokens, etc.) and the app will extract hex codes and render a palette automatically.

## Getting started

No setup required.

1. Open `index.html` in your browser.
2. Paste hex codes into the text area.
3. The palette updates instantly with any number of colors.

### Example input

```
{
  "brand": "#1E90FF",
  "accent": "#fef08a",
  "neutral": "#111827",
  "alpha": "#22c55e80"
}
```
